1. Requirement: Visual Studio 2008 with SP1 and Python 2.5 or 2.6 installed.

2. Enter ..\proj and modify "VCBUILD_PATH" in Make.bat to identify the path of "vcbuild.exe" of VS 2008.

3. run Make.bat, then the generated executables will be in ..\bin\Release.

4. The solution can also be opened and compiled by proj\NetShield.sln.

5. The test sample is in test\.
